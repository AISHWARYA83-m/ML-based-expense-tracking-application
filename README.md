ML Based Expense Tracking Application

Overview
This project is an ML-based expense tracking application that helps users analyze and manage their expenses efficiently.

Features
- Expense tracking
- Expense category prediction
- Monthly spending analysis
- Interactive dashboard
- Data visualization using charts

Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib

Project Structure

Smart_Expense_Tracker/
│── app.py
│── requirements.txt
│── dataset.csv
│── README.md


 Installation

pip install -r requirements.txt

 Run the Project

streamlit run app/dashboard.py
